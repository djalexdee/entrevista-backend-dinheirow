# Exercício Node / Express / MongoDB/PostgreSQL

Antes de começar a entrevista técnica, você deve prepara seu ambiente local para o funcionamento correto da aplicação. Instale os pacotes e execute o script `npm run dev` para iniciar o servidor em modo desenvolvimento. Recomendamos também que dê uma olhada no código para se familiarizar com a estrutura.

# Como será a entrevista técnica?

Nossa sessão será como um "pair programming", iremos pedir a implementação de uma ou mais funcionalidades, a correção de bugs ou ideias de arquitetura/refatoração. Você poderá usar sites, consultar documentações ou usar IA como apoio durante a sessão. Entraremos em mais detalhes durante nossa sessão de código, nos vemos em breve!

# Requisitos básicos

-   Sua IDE/Editor de preferência
-   NodeJs > 24.X
-   Algum programa para requisições à API, como cURL, Postman, Insonmia...

## Bibliotecas usadas:

-   Routing Controllers (https://github.com/typestack/routing-controllers)

## Rotas:

-   Voos => /v1/flights

Lembre-se de preparar seu ambiente de desenvolvimento antes da entrevista. Você poderá usar MongoDB ou PostgreSQL mudando a variável de ambiente DB_TYPE no arquivo .env.
